# unity-township-simulation
Unity C# project showcasing association relationships in object-oriented programming.

## Project Overview
This project demonstrates modular, object-oriented simulation using C#.

The application dynamically generates a "Township" composed of multiple "Villages," each containing a variable number of "Houses." The project features procedural instantiation, constructor-based initialization, and class composition.

## How to Run
1. Download or clone the repository  
2. Open Unity Hub and click "Add" to add the project folder
3. Project should open in 2D mode
4. Press the **Play** button in the Unity Editor to run the simulation
